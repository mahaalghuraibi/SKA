# Custom food dataset layout

Place training images under **`raw/<class>/`** using exactly these folder names:

| Folder   | Meaning        |
|----------|----------------|
| `fish`   | Fish / seafood |
| `chicken`| Poultry        |
| `meat`   | Red meat cuts  |
| `kebab`  | Kebab / kofta / skewers |
| `pasta`  | Pasta / noodles |
| `salad`  | Salads         |
| `rice`   | Rice dishes    |
| `soup`   | Soups / stews (liquid-forward) |
| `bread`  | Bread / flatbread |

Use **JPEG or PNG**, varied lighting and angles. See the parent **`../README.md`** for collection guidelines and how to run training.
